# 🚀 Skill Exchange Marketplace

A modern Python-based web application where users can exchange, manage, and showcase their skills online.

---

## 🌟 Features

- 👤 User Registration & Login
- 🛠️ Skill Management
- 📁 Profile Upload
- 🤝 Skill Exchange System
- 🔍 User Search
- 📄 Signature Preview
- 💾 Database Integration
- 🔐 Secure Authentication

---

## 🧰 Technologies Used

- Python
- Flask
- HTML5
- CSS3
- SQLite

---

## 📂 Project Structure

```bash
├── app.py
├── models.py
├── forms.py

├── uploads/
└── requirements.txt
```

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/skill-exchange-marketplace.git
```

### 2️⃣ Open Project Folder

```bash
cd skill-exchange-marketplace
```

### 3️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Project

```bash
python app.py
```
